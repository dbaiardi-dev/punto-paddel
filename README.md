# 🎾 PuntoPaddel

Contador de puntos de pádel con detección de gestos. Marcá tus puntos sin tocar el celular.

## ✨ Features

- **Contador de puntos automático** - Reglas oficiales de pádel
- **Detección de gestos** - Levantá la mano 3 segundos para marcar punto
- **UI de cancha** - Pantalla dividida estilo cancha de pádel
- **Estadísticas** - Historial de partidos jugados
- **Premium de por vida** - $3 USD para early adopters

## 🎮 Cómo usar

1. Tocá "Comenzar Partido" en el dashboard
2. Tocá la mitad del equipo que anotó, o...
3. Activá la cámara y levantá la mano 3 segundos para marcar punto
4. ¡Jugá!

## 🏗️ Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Web Camera API (para gestos)

## 🚀 Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
